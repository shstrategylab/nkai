📋 MAKE.COM 무료 자동화 설정 가이드 (10분 설정, 평생 무료)
Step 1 — make.com 가입 (무료) → [Create a scenario] 클릭
Step 2 — 첫 모듈: Webhooks → Custom webhook 추가 → URL 복사 → 위 입력란에 붙여넣기
Step 3 — HTTP 모듈 추가: Yahoo Finance VIX
   URL: https://query1.finance.yahoo.com/v8/finance/chart/%5EVIX?interval=1d&range=1d
   → Parse JSON → 추출: chart.result[0].meta.regularMarketPrice
Step 4 — 동일하게 WTI(CL%3DF), 통일부 RSS, 외교부 RSS 모듈 추가
Step 5 — 마지막 모듈: Webhooks → Webhook response
   Body에 JSON 조합: {"vix": {{vix}}, "wti": {{wti}}, "nk_news": "{{rss_summary}}"}
Step 6 — [Schedule] → 매일 오전 9시 자동 실행 설정

Make.com 무료 플랜: 1,000 ops/월 (매일 실행 시 ~300 ops/월 소모). CSI는 한국은행 ECOS API 무료 키 발급 후 동일 방식 추가 가능.
본 현황판은 상수학적 프레임과 현대 데이터 지표를 결합한 개인 분석 도구입니다.
완전 자동: 환율(Open Exchange Rates CORS OK) · 태양흑점(NOAA SWPC CORS OK)
Make.com 웹훅: VIX · WTI · CSI · RSS뉴스 → 서버사이드 수집 후 이 페이지로 전달 (CORS 우회)
AI 분석: GDELT 무료 API + Claude → 北도발 · 외교 · 분쟁강도 · 발언톤 · 간극 자동 추출
수동 유지: 외국인순매수(KRX) · 수출증감률(월간)
